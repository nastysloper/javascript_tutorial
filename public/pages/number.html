Numbers

In the console, type 5 + 5 and hit enter.

Pretty easy, right?

Try some more operations.

Here's something different: type 100 % 5 and press enter.

% is called the modulus operator.

The modulus operator performs division but returns the remainder.

Try 5 % 4 and see what you get.

Now, type parseInt("2") and press return. Don't forget the quotes.

parseInt( ) takes a number as a string and returns the actual number.
