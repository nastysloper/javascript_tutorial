Strings

A string is a sequence of characters, like a word or sentence.

Type your name in the console. If you leave off the quotes, you'll get an error:

ReferenceError: <your_name> is not defined

Type your name, in quotes, followed by .length and press return

For example, "Mack".length

Then, type "Mack".charAt(0)

Type "Mack" + "truck"

Putting Mack and truck together is called string concatenation.
