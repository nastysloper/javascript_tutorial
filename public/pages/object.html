Objects

An object in JavaScript can be thought of as a collection of name-value pairs

Similar structures, called hashes, exist in other languages, such as C, Java, and Ruby.

The "name" part is a string, and the value can be any value, including other objects.

This allows you to build data structures of arbitrary complexity.


There are two ways to create an object:

 var obj = { }

and

  function Person(name, age)
  {
    this.name = name;
    this.age = age;
  }

Now create a new instance of a person object:

 var you = new Person("Mack", 30);
