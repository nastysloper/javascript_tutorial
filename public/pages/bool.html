Boolean

A boolean is a value that is either true or false.

false, 0, an empty string "", NaN (not a number), null, and undefined are all false.

all other values evaluate to true.


The boolean operators include && (and), || (or), and ! (not).

Try typing in the following in the console:

  • 1 || 0
  • 1 && 0
  • 1 || 1
  • 1 && 1
  • ! 0

In computer logic, true or false evaluates to true. true and false evaluates to false.
